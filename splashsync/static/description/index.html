Splash Sync Module for Odoo

Splash Php Module for Odoo ERP Platforms.

This module implement Splash Sync connector for Odoo ERP. It provides access to multiples Objects for automated synchronization though Splash Sync dedicated protocol.

Installation

For all install instructions, see: Odoo Module Doc

Requirements

  • Python 3.9+
  • An active Splash Sync User Account
